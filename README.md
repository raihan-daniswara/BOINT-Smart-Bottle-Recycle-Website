# BOINT – Smart Bottle Recycle Website

## Project Overview

**BOINT (Bottle Point)** adalah platform website daur ulang pintar yang dirancang untuk mendorong masyarakat mendaur ulang botol plastik melalui sistem poin digital.

Platform ini mengintegrasikan **mesin pengumpul botol berbasis IoT** dengan **aplikasi web**, yang berfungsi untuk:

* mencatat setoran botol
* mengelola poin pengguna
* memungkinkan penukaran poin dengan hadiah

Proyek ini bertujuan meningkatkan kesadaran lingkungan dan mendorong perilaku ramah lingkungan dengan mengubah limbah botol plastik menjadi **poin yang dapat ditukarkan dengan berbagai reward**.

---

## Tujuan Project
* Mengurangi limbah botol plastik
* Meningkatkan kesadaran masyarakat terhadap daur ulang
* Memberikan insentif digital melalui sistem poin
* Mengintegrasikan teknologi IoT dengan platform web
* Mendukung program sekolah ramah lingkungan **(Adiwiyata)**

---

## Fitur Utama

### Fitur Pengguna
* Registrasi dan login pengguna
* Manajemen profil pengguna
* Melihat riwayat setoran botol
* Mendapatkan poin dari botol yang disetorkan
* Menukarkan poin dengan reward
* Melihat riwayat penukaran reward

### Fitur Admin
* Manajemen data pengguna
* Monitoring perangkat mesin BOINT
* Monitoring data setoran botol
* Manajemen reward
* Validasi penukaran reward
* Dashboard statistik penggunaan sistem

### Integrasi IoT
* Pairing perangkat dengan akun pengguna
* Pengiriman data botol dari mesin ke server
* Sinkronisasi data secara real-time
* Monitoring status perangkat
